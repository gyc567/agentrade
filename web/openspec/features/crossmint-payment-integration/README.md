## ❓ 常见问题

### Q1: 支付时出现 "钱包连接失败" 怎么办？

**解决方案**:
1. 确保你的钱包扩展已安装在浏览器中
2. 检查钱包是否已解锁（需要输入密码）
3. 尝试刷新页面重新开始
4. 尝试使用 WalletConnect 而不是 MetaMask
5. 清除浏览器 Cookie 后重试

### Q2: 我选择了错误的区块链怎么办？

**不用担心**！Crossmint 支付窗口允许你在支付前更改区块链链。只需在支付窗口中选择不同的链。

### Q3: 支付失败了，我的钱去哪了？

**安全保证**：如果交易失败，你的钱不会被扣除。
- 区块链会自动回滚交易
- 你可以安全地重新尝试
- 我们无法访问你的资金

### Q4: 需要多长时间才能收到积分？

```
时间线：
┌────────────────────────────────────┐
│ 1. 钱包确认: 5-30 秒               │
│ 2. 区块链打包: 10-60 秒             │
│ 3. 链上确认: 30 秒 - 2 分钟         │
│ 4. 积分到账: 立即                  │
├────────────────────────────────────┤
│ 总耗时: 1-5 分钟（通常）          │
└────────────────────────────────────┘
```

### Q5: 有交易费用吗？

**价格透明**：
```
你支付: USDT 数量（显示在支付窗口）
包括: Crossmint 处理费用（~1%）
不包括: Gas 费用（由区块链收取，显示单独）
```

### Q6: 支付历史在哪里看？

进入 **个人资料** → **支付历史** 可以看到：
- 所有过去的购买
- 购买金额和积分
- 支付状态
- 交易哈希（用于区块链验证）

### Q7: 积分可以退款吗？

**政策**：
```
✅ 支付失败 → 自动全额退款
❌ 支付成功 → 不退款（积分已加入账户）
```

一旦区块链确认，交易无法撤销。这是区块链的特性，不是我们的限制。

### Q8: 支持哪些国家？

**99% 的国家都支持**，除了：
- 🚫 朝鲜
- 🚫 伊朗
- 🚫 叙利亚
- 🚫 克里米亚

**如果你在上述地区，无法使用此功能**。

## 🚨 故障排查

### 支付窗口加载很慢？

```
✓ 刷新页面
✓ 清除浏览器缓存
✓ 尝试不同的浏览器
✓ 检查网络连接
✓ 禁用 VPN（如果使用）
```

### 钱包连接失败？

```
MetaMask 用户:
  1. 打开 MetaMask 扩展
  2. 检查是否已解锁
  3. 检查网络选择（应为 Polygon/Base/Arbitrum）
  4. 点击账户，确认已连接

WalletConnect 用户:
  1. 打开你的钱包应用
  2. 打开 WalletConnect 功能
  3. 扫描二维码或复制链接
  4. 在钱包应用中批准连接
```

### 交易一直待处理？

```
等待时间表:
  0-30 秒:   钱包确认中
  30s-2min:  区块链处理中（正常）
  2min+:     可能有问题

如果超过 5 分钟:
  1. 检查你的钱包（交易可能已确认）
  2. 到区块链浏览器搜索你的钱包地址
  3. 查看交易状态
  4. 联系支持
```
